putils.util.edit_distance(seq1, seq2, method='levenshtein')[source]¶

두 iterable sequence의 edit distance를 반환한다.

iterable seq1, seq2 사이의 edit distance를 반환한다. seq1, seq2가 동일 object이거나 값이 같다면 0을 반환한다. 현재 method는 levenshtein만 지원한다.

Parameters:
  • seq1 (Iterable) – edit distance를 구할 iterable
  • seq2 (Iterable) – edit distance를 구할 iterable
  • method (str) – edit distance를 구할 method. 현재는 levenshtein만 지원.
Returns:

seq1과 seq2의 edit distance
Raises:

ValueError – seq1, seq2 둘 중의 하나가 iterable이 아니거나, method가 올바른 값이 아닌 경우.
