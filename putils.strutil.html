putils.strutil.get_json_value(json_obj, element_name)[source]¶

json_obj에서 element_name의 value를 가져온다.

기본적으로 json_obj[element_name] 값을 반환한다. 이 때, element_name이 “/”로 나뉘어져 있으면, 하위 element name으로 생각한다. 즉, “parent/child” 로 이름이 들어왔으면, json_obj[“parent”][“child”]를 반환한다.

Parameters:
  • json_obj – 살펴볼 json object.
  • element_name – 값을 가져올 element name. ‘/’는 parent/child 구분자로 사용한다.
Returns:

json_obj에서 element_name의 value. element_name이 없다면 KeyError.

putils.strutil.str_join(data_list, sep='\t')[source]¶

data_list들을 하나의 문자열로 만든다.

str.join()과 유사하게 동작한다. 단, data_list가 str이 아닌 경우에도 사용 가능하다.

Parameters:
  • data_list – join해서 하나로 만들 데이터 리스트.
  • sep – data_list의 element들을 join할 때 사용할 glue str.
Returns:

sep로 data_list를 붙인 하나의 str.
