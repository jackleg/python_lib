putils.dfutil.find_by_value(df, column_name, value, first=True)[source]¶

df의 column에 있는 특정 값의 위치를 찾는다.

Parameters:
  • df – 값을 찾을 DataFrame
  • column_name – 값을 찾을 column name(df[column_name])
  • value – 찾을 값.
  • first – 동일한 값이 여러 개 있는 경우 처음 나타나는 값을 반환할 지 여부. False이면 마지막 값을 반환.
Returns:

df[column_name] == value인 index. 해당하는 값이 없다면 None.
putils.dfutil.has_null(ser, need_count=False)[source]¶

series에서 null value가 있는지 여부 확인.

Parameters:
  • ser – 확인할 값이 있는 Series.
  • need_count – null 값이 몇 개나 있는지 값을 함께 반환할 지 여부.
Returns:

ser에 null value가 있으면 true 없다면 false. need_count가 True라면 (null value 여부, null value 개수)의 pair.
